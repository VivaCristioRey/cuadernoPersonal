NodeJS

1. ¿Qué es NodeJS?

Es un entorno de ejecución para JavaScript

2. Diferencias entre Javascript en el frontend y Javascript en el navegador

En el Frontend sirve para la interactividad del cliente con el aplicativo web, y en el navegador se usa para ejecutar scripts para distintos objetivos

3. Ventajas y desventajas del uso de Javascript en el Backend

Como ventajas encontramos que es un lenguaje que se presta para hacer cualquier cosa, por lo tanto es usado por muchas empresas, entre ellas empresas reconocidas como La Nasa, Netflix, LinkedIn, Paypal, etc... No es complicado de usar ya que no es un lenguaje alto tipado. Por otro lado, como desventajas encontramos que este consume muchos recursos

4. Paso a paso para la instalación y configuración

Windows:

1. Buscaremos en internet nodejs.org e ingresaremos al primer link

2. Entramos y descargamos NodeJs (se recomienda descargar la LTS (Long Term Support))

3. Seleccionamos la ruta donde se descargará el instalador de NodeJs y le damos a Guardar

4. Ejecutamos el instalador y nos saldrá una ventana para seguir los pasos de instalación (Asegúrate de que esté marcada la opción: Add To PATH finalizando la instalación), y terminas de instalar

5. Abrimos la terminal de Windows y escribiremos "node -v" y "npm -v" para verificar la versión
